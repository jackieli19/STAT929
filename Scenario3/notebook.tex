    \subparagraph{2.KNN}\label{knn}

    \begin{itemize}
\tightlist
\item
  This method uses k-nearest neighbor to fInd similar samples and
  imputed unobserved data by weighted average of similar observations.
\item
  Basic idea: Impute array with a basic mean impute and then use the
  resulting complete array to construct a KDTree. Use this KDTree to
  compute nearest neighbours. After finding k nearest neighbours, take
  the weighted average of them. Basically, find the nearest row in terms
  of distance
\end{itemize}
